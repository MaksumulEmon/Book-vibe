# 📚 Book Vibe


Book Vibe is a modern web application that helps users explore and discover their next favorite books in a simple and engaging way. It provides a smooth user experience with a clean and responsive design.

🚀 Live Demo
👉 https://book-vi-be.vercel.app/


## ✨ Features
 📖 Browse and explore book collections
 🔍 Easy and smooth navigation
 🎯 Clean and user-friendly UI
 📱 Fully responsive design
 ⚡ Fast performance

# 🛠️ Technologies Used
HTML
CSS / Tailwind CSS
React.js


# 📦 Installation
Follow these steps to run the project locally:

## Clone the repository
git clone Your Repo Link Here

## Go to project folder
cd book-vibe

## Install dependencies
npm install

## Run the project
npm run dev
